Comprehensive Java Programming

Duration

Course Contents

This course is designed for beginners in which they will learn how to create their own applications and games by using Java language with little experience in programming. This Mater course will help them to start their professional journey as a “Confident Java Programmer”.

Here’s more detail about what they’ll learn:

  • Gain in-depth understanding of core & advanced features of Java including JVM internals.
  • Get real-world experience by developing an instructor-led Java EE-based Web application using technologies like Servlets and MySQL
  • Master Object-Oriented Programming concepts by using a real-world application as a case study
  • Solid understanding of Functional-style Programming using Java 8 constructs like Lambdas & Streams
  • Master design principles, best practices and coding conventions for writing well-designed, professional Java code
  • Set firm foundation in Java for the rest of your career. That's a promise!
  • Master Exceptions, IO, Collections Framework, Generics, Multi-threading, Databases, etc. with best practices
